2010有道难题资格赛第三轮解题报告
2010-05-31 00:59

选课的困惑
表达式变型为
(成绩[i]-90)*学分[i]
求和大于等于0.
i从1到n已经确定，不妨设为value.
题目意思就是说从剩下的m个中选最少的个数出来，要使和value得加起来大于等于0.
很明显把剩下的m个的(成绩[i]-90)*学分[i]从大到小排序，一个一个地加，加到大
于等于0或者没有加的为止。

X星球的身份证系统
注意读题，身份证的长度是n。
n是偶数则找出比前一半小的数个数。
然后对于刚到取前一半的时候，只需要验证前一半翻转后是不是小于等于后一半。
对于n是奇数，类似。

火柴游戏
dp[i][j]表示i根火柴组成的模m余j的数目。
设组成x需要need[x]的火柴数，于是
for (int i = 0; i < n; ++i)
for (int j = 0; j < m; ++j) if (dp[i][j] != 0)
for (int k = 0; k < 10; ++k)
if (i + need[k] <= n)
{
int& u = dp[i + need[k]][j*10 + k) % m];
u += dp[i][j];
if (u >= mod) u -= mod;
}
直接dp完再处理就行了。
注意到不能为首零，所以初始化的时候：
for (int i = 1; i < 10; ++i)
++do[need[i]][i % m];
dp完之后n>=6的话dp[need[0]][0]再加1就行了。

在开数组上的trick是
inline int& at(int x, int y) {return dp[x*m + y];}
